IOT Based Gas Leakage Detection System

A major project developed using Arduino, Embedded C, Python, and IoT technologies for detecting gas leakage and fire hazards in household environments.

Features

* LPG gas leakage detection
* Fire detection system
* Automatic buzzer alerts
* Exhaust fan activation
* Water sprinkler activation
* LCD status display
* IoT-based monitoring webpage

Technologies Used

* Arduino Uno
* Embedded C
* Python
* ESP8266 Wi-Fi Module
* MQ-5 Gas Sensor
* Flame Sensor

Project Overview

The system continuously monitors gas leakage and fire hazards using sensors connected to an Arduino microcontroller. When danger is detected, the system activates safety mechanisms including a buzzer, fan, and water sprinkler while also displaying alerts on an LCD screen and IoT webpage.
